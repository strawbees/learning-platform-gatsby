Create a Robot RacerBuild a robot that walks and then try to make it run.

Creating a robot that walks around is a great activity to gain basic building and language skills. By placing the motor on unexpected places, switching the speed it moves as well as upgrading your Strawbees construction and placing carefully the Quirkbot, you can get many styles of movement. We recommend uploading the Factory Program to your Quirkbot before the activity.

Organising the robots on a "race" like situation is a suggestion and can be perfectly replaced by other challenge or constraint.

Running in a straight line can be challenging

Once you are able to observe and describe the different types of motor movements, it's easier to describe how else it should move. Use our CODE programming interface to translate it to the Quirkbot.

Quirkbot works on Google Chrome Browser and ChromeOS
